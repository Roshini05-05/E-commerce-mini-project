# E-commerce-mini-project

This E-commerce mini project is developed using JavaScript, focusing on client-side interactions and basic product management.

The application allows users to browse items, view product details, and add products to a shopping cart. JavaScript is used to dynamically update the UI, handle user actions, and store cart data using localStorage.

The project provides a simplified version of online shopping to demonstrate core E-commerce functionalities using only frontend technologies. It is ideal for beginners to understand DOM manipulation, event handling, and basic web application flow.

Features:
1.Displays a list of products with name, price, and image

2.Allows users to view detailed product information

3.Add-to-cart functionality for each product

4.View cart page showing selected items

5.Remove items from the cart

6.Update product quantity in the cart
